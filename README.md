# 🚗 Análisis de Venta de Vehículos en EE. UU.

## Descripción del Proyecto
 "Esta es una aplicación web interactiva construida en Python que permite a los usuarios analizar un conjunto de datos de anuncios de venta de coches..."

## Funcionalidades
- Limpieza de datos y manejo de valores nulos.
- Generación de un histograma interactivo para analizar la distribución de precios.
- Gráfico de dispersión para explorar la relación entre el precio y el kilometraje de los vehículos.

## Herramientas Tecnológicas
- **Python**
- **Pandas** 
- **Plotly Express**
- **Streamlit** 

## Cómo ejecutar este proyecto localmente
1. Clona este repositorio en tu máquina local.
2. Asegúrate de tener instaladas las librerías necesarias (`pandas`, `plotly-express`, `streamlit`).
3. Ejecuta el siguiente comando en tu terminal:
   
   streamlit run app.py
